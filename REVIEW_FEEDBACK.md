## 二、严重问题：减号渲染为⊗

### 问题描述

源码 `generate_ch2_ch3_foundation_pdf.py` 第186行写的是：

```
σ_a = (σ_max − σ_min)/2
```

其中 `−` 是 Unicode U+2212（数学减号）。但 PDF 实际渲染出来变成了：

```
σ_a = (σ_max ⊗ σ_min)/2
```

`⊗` 是 U+2297（张量积符号），对初学者会造成严重误导。

### 受影响位置

| 页码 | 公式 | 问题 |
|------|------|------|
| p.7 | `σ_a = (σ_max ⊗ σ_min)/2` | 应为减号 |
| p.7 | `r = ⊗1`（对称循环） | 应为 `r = −1` |
| p.10 | 速查卡中同样公式 | 同上 |

### 根因分析

脚本使用 `UnicodeCIDFont('STSong-Light')` 注册字体（第15行）。该 CID 字体在 CMap 映射中，U+2212 的字形被错误映射到了 ⊗ 的码位。

### 修复方案

**方案A（简单，推荐先做）：** 把源码中所有 `−`（U+2212）替换为普通连字符 `-`（U+002D）：

```python
# 在 generate_ch2_ch3_foundation_pdf.py 顶部加一行全局替换
import re
# 读取文件后统一替换
text = text.replace('\u2212', '-')
```

**方案B（彻底，效果最好）：** 改用 matplotlib mathtext 渲染公式为印刷体 PNG 图片嵌入 PDF。参考 `考研数学二_由极限定义的函数及其间断点判定.pdf` 的做法：

```python
import matplotlib.pyplot as plt
fig = plt.figure()
fig.text(0.5, 0.5, r'$\sigma_a = \frac{\sigma_{max} - \sigma_{min}}{2}$', fontsize=16)
fig.savefig('formula.png', dpi=200, bbox_inches='tight', transparent=True)
# 嵌入 reportlab Image
```

方案B能同时解决「减号错误」和「公式非印刷体」两个问题。

## 三、需要补强的部分

### 3.1 公式印刷体排版

当前公式是纯文本放在灰底框里，下标用 `_` 表示（如 `σ_max`、`σ_min`），不是真正的数学排版。

建议：关键公式至少用 matplotlib mathtext 渲染为图片，效果如下对比：

| 当前效果 | 印刷体效果 |
|---------|-----------|
| `σ_N^m N = C` | σ 的 N 次方乘 N 等于 C，真正的上标 |
| `σ_a = (σ_max − σ_min)/2` | 真正的分数线、下标 |

### 3.2 课本图片引用

三本教材 PDF 里有大量插图（疲劳断口照片、S-N曲线图、磨损三阶段曲线、摩擦状态示意图等），但讲义中一张都没有引用。

建议：用 `pdf2image` 或 `PyMuPDF` 从教材 PDF 中提取关键图片：

```python
import fitz  # PyMuPDF
doc = fitz.open('mechanical_design_textbook.pdf')
page = doc[25]  # 第26页
pix = page.get_pixmap(dpi=150)
pix.save('fig_2_5_sn_curve.png')
```

每个知识点至少引用1张课本原图，标注：
> 📷 [课本图 2-5：S-N曲线] —— 横轴是循环次数N（对数坐标），纵轴是应力水平σ，曲线表明应力越低可承受循环次数越多。

### 3.3 推导深度

部分知识点的「一步一步推导」偏简略。以知识点2.5为例：

当前写法：
```
第2步：把交变部分乘以K_σ，考虑真实零件削弱。
```

建议补充数值演示：
```
第2步：把交变部分乘以K_σ。
  例：σ_a = 100 MPa，K_σ = 1.5（有键槽），
  则等效应力幅 = 1.5 × 100 = 150 MPa。
  这说明键槽让危险程度提高了50%。
```
